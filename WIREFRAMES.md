## Design Notes

### Color Scheme
- **Primary Color:** Purple (#512BD4)
- **Secondary Color:** Light Gray
- **Error Text:** Red
- **Background:** White
- **Text:** Charcoal/Dark Gray

### Typography
- **Headlines:** 32px, Bold
- **Section Titles:** 18-24px, Bold
- **Body Text:** 14-16px, Regular
- **Labels:** 12-14px, Regular

### Component Guidelines
- **Buttons:** Rounded corners (8px), minimum height 44px
- **Cards/Frames:** Light shadow, rounded corners, border
- **Input Fields:** Clear labels, placeholder text, validation feedback
- **Lists:** Spacing between items (8px), interactive items clearly indicated
- **Loading States:** Activity indicators or skeleton screens
- **Empty States:** Helpful messages guiding users to take action

### Navigation Patterns
- **Bottom Tab Bar:** Primary navigation (Terms, GPA, Search, Financial)
- **Back Button:** Top-left for secondary pages
- **Action Buttons:** Top-right (Add, Edit actions)
- **Modal Forms:** Slide-in or overlay for edit/create forms

### Responsive Considerations
- Wireframes shown for mobile/tablet portrait orientation
- Desktop/Windows version adapts to larger screen with multi-column layouts
- Minimum touch target size: 44x44px
- Scrollable content areas for long lists
